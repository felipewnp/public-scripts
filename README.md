# Available Scripts

## Bitbucket
- [Get Bitbucket Repository Variable](bitbucket/get-bitbucket-repository-variable/README.md) - Retrieves usage variables from Bitbucket repositories.

# Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/felipewnp/public-scripts.git
    cd public-scripts
    ```
2. Ensure you have the necessary permissions (e.g., executable rights) for the scripts you intend to run.
    ```bash
    chmod +x path/to/script.sh
    ```
